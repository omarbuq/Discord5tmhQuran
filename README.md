# Discord5tmhQuran
بوت يرسل كل بعد صلاة 5 صفحات من القران لختم القران في نهاية رمضان

يجب عليك في index.js إضافة id الروم + التوكن للبوت

- عط صلاحيه للبوت للمنشن والكتابه وإرسال 

طريقة التشغيل

npm install discord.js node-schedule adhan

بعدها شغل البوت بـ
- node index.js

لاتنسونا من صالح دعائكم
